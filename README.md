# 📚 BookLog (북로그) - 개인 독서 기록 및 추천 서비스

대학 SW프로그래밍 기말 대체 과제로 개발된 **BookLog(북로그)**는 개인의 독서 활동을 직관적으로 기록하고 관리하며, 유용한 추천 도서를 탐색할 수 있는 웹 기반 개인 서재 관리 애플리케이션입니다.

---

## 🌟 프로젝트 소개
* **개발 형식**: 웹 프론트엔드 단독 프로젝트 (순수 Vanilla HTML, CSS, JavaScript)
* **목적**: 데이터베이스나 서버 구축 없이 브라우저의 `localStorage`를 최대로 활용하여 실시간 데이터 영속성 및 개인화된 다중 사용자 프로필 격리 환경을 구축.
* **디자인 테마**: 모던하고 세련된 글래스모피즘(Glassmorphism) 레이아웃 및 다크/라이트 테마 즉시 전환 기능 내장.

---

## 🚀 주요 기능

### 1. 개인화된 다중 프로필 관리 게이트
* **개인 데이터 격리**: 사용자의 **이름**과 **생년월일**을 입력받아 `booklog_books_[이름]_[생년월일]` 키값으로 데이터를 개별 관리합니다. 
* **프로필 목록**: 로컬 브라우저에 등록된 기존 프로필들을 감지하여 로그인 화면에서 '간편 프로필 선택' 버튼을 통해 즉시 로그인하고 서재 데이터를 전환할 수 있습니다.
* **샘플 데이터 옵션**: 최초 로그인 시 "샘플 독서 데이터 로드하기"를 선택하면 바로 대시보드 통계와 서재 기능을 테스트할 수 있는 샘플 도서 5권이 자동으로 주어집니다.

### 2. 독서 통계 대시보드 (Dashboard)
* **핵심 요약 카드**: 다 읽은 책 수, 읽고 있는 책 수, 평균 평점 별점 등을 요약하여 보여줍니다.
* **순수 커스텀 SVG 독서 그래프**: 외부 무거운 그래프 라이브러리(Chart.js 등) 없이 **순수 SVG 태그와 연산**으로 직접 드로잉한 독서 통계 막대그래프를 제공합니다. 마우스 호버 시 상세 권수가 툴팁으로 나타납니다.
* **최근 도서 타임라인**: 가장 최근 서재에 추가되거나 읽기 시작한 도서 3권의 상태를 즉시 확인할 수 있습니다.

### 3. 나의 서재 (Library)
* **실시간 도서 검색**: 도서 제목 또는 저자 이름 검색어를 타이핑하는 즉시 서재 목록이 필터링됩니다.
* **도서 상태 필터**: '읽기 전', '읽는 중', '완독' 등 독서 상태에 맞는 도서들만 모아서 볼 수 있습니다.
* **정렬 시스템**: 최근 등록순, 제목 가나다순, 별점 높은순, 읽기 시작일 순 정렬 기능을 제공합니다.
* **상세 조회 및 삭제**:
  * 팝업 모달을 통해 도서 줄거리 및 감상평을 한눈에 조회합니다.
  * 실수로 인한 데이터 유실을 방지하기 위해 브라우저 기본 alert 대신 **자체 디자인된 2단계 확인(Double-Check) 삭제 폼**을 탑재했습니다.

### 4. 도서 등록 및 수정 모달
* **별점 인터랙션**: 1점부터 5점까지 별 아이콘에 마우스를 대거나 클릭하는 시각 인터랙션을 지원합니다.
* **유동적인 폼 입력**: 독서 상태가 '읽기 전'일 경우 불필요한 평점과 감상평 입력란을 자동으로 숨겨 깔끔한 인터페이스를 유지합니다.
* **직접 작성 줄거리**: 도서 추가/수정 시 책의 핵심 줄거리를 사람이 직접 입력하고 수정할 수 있습니다.

### 5. 선별 추천 도서 (Recommendations)
* **계층 구조 카테고리**: '문학(소설, 시/희곡, 에세이)'과 '비문학(경제/경영, 자기계발, 과학/IT, 인문사회 등)' 2단계 구조의 명저 아카이브 **총 90권**이 내장되어 있습니다.
* **이달의 추천 도서 (자동 롤링)**: 현재 날짜 정보의 '월(Month)' 데이터를 시드로 삼아, 매달 자동으로 다른 분야의 엄선된 도서 3권이 '이달의 추천 도서' 카드에 동적으로 로테이션 노출됩니다.
* **내 서재에 담기**: 추천된 도서 마음에 들면 클릭 한 번으로 내 서재에 '읽기 전' 도서로 즉시 추가할 수 있습니다.

## 🛠 기술 스택 & 개발 환경
* **Frontend**: HTML5, Vanilla JavaScript, CSS3 (Vanilla CSS)
* **디자인 및 레이아웃**: CSS Variables 정의 기반 네온 글로우 스타일링, 반응형 그리드 시스템 레이아웃 (PC, 태블릿, 모바일 모두 최적화 지원)
* **빌드 및 배포 보조**: Vite (경량 빌드 환경 제공 및 GitHub Pages 배포 지원)

## 💻 실행 방법

### 1. 로컬에서 단순 실행하기
이 애플리케이션은 순수 웹 표준 파일들로 구성되어 있어 별도의 서버 실행 없이 바로 동작합니다.
1. 이 프로젝트 폴더 안의 `index.html` 파일을 웹 브라우저(Chrome, Safari 등)로 직접 엽니다. (더블 클릭)

### 2. Vite 개발 서버를 이용해 실행하기
1. 프로젝트 폴더 경로에서 터미널을 열고 패키지를 설치합니다.
   ```bash
   npm install
   ```
2. 로컬 개발 서버를 구동합니다.
   ```bash
   npm run dev
   ```
3. 터미널에 표시되는 로컬 서버 주소 뒤에 설정된 기본 배포 패스인 `/booklog/`를 덧붙여 접속합니다.
   * 예: `http://localhost:5173/booklog/`
